# Тестовый сценарий: Поступление в "Школу 21"

## 1. Общая информация
- ID сценария: 03 
- Название: Проверка процесса поступления в Школу 21 через страницу «Как поступить»
- Раздел: Сайт - Раздел «Поступающим» - Подраздел «Как поступить»

## 2. Предусловия
- Сайт школы доступен
- Пользователь старше 18 лет
- Идет набор студентов

## 3. Основные шаги
1. **Регистрация на сайте**
   - Перейти по кнопке "Поступить в школу"
   - Проверить корректность перехода на форму заявки
   - Заполнить форму валидными данными ( Кампус, программа, эл. почта(при входе без SberID))
   - Отметить согласие с обработкой персональных данных
   - Нажать кнопку "Поступить"

2. **Онлайн-тестирование**
   - Пройти логические игры (1 час)
   - Получить результаты

3. **Запись на "бассейн"**
   - Выбрать дату отборочных испытаний
   - Подтвердить участие

4. **Прохождение бассейна**
   - Посещение 2-недельного интенсива
   - Выполнение ежедневных индивидуальных проектов (7)
   - Выполнение 2-ух групповых проектов

5. **Получение результата**
   - Проверить статус в личном кабинете или на почте
   - Получить финальное решение

## 4. Ожидаемые результаты
- После регистрации: письмо с подтверждением на email
- После тестирования: приглашение на следующий этап (при успехе)
- После бассейна: статус "Зачислен" в личном кабинете

## 5. Альтернативные сценарии
- **Не сдал онлайн-тест:** Предложение попробовать заново
- **Не прошел бассейн:** Возможность повторной попытки в следующем наборе

## 6. Критерии успеха
- Все этапы выполняются без технических ошибок
- Уведомления приходят своевременно
- Статусы в системе обновляются корректно
